# Tales of @

This is a terminal based game with a 2d height based procedural map generation

## What's Included

Day and night Cicle
4 diferent types of enemys
WASD movement and Arrows Firing bullets

## Getting Started

Once you enter the game you will see something like this 
```
WASD = walk
Arrows = fire bullets

@ = Your character
# = Walls
" and ^ = Grass
~ and - = water

C = Enemys that Chases player
F = Enemys that Flee player
B = Enemys that Flee bullets
D = Enemys that Flee bullets and Chases player
```
